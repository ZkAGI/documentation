# ZkAGI x Solana Agent Kit

## How does ZkAGI integrate with the Solana Agent Kit? {#how-does-zkagi-integrate-with-the-solana-agent-kit}

<p className="snippet-answer">ZkAGI is the first to integrate a decentralized API alternative to OpenAI on the Solana Agent Kit, enabling verifiable and decentralized AI powers in agents. It supports the widest selection of open-source AI models and LLMs with proof of sentience via Zk proofs.</p>

- ZkAGI becomes the first to integrate a **decentralized API alternative to OpenAI** on Solana Agent Kit
- Integrate the widest selection of **open-source AI models and LLMs**
- Operate agents with **proof of sentience** — Zk proofs to verify AI output and the AI having a "mind" of its own
- Integrate and expand with **Eliza and other agentic frameworks** to craft the agent of your dreams

- Github Code: [ZkAGI Agent Kit](https://github.com/ZkAGI/zkagi-agent-kit/tree/main)
- Report bugs or suggestions to: [core@zkagi.ai](mailto:core@zkagi.ai)

## What can agents do with the ZkAGI Solana Agent Kit? {#what-can-agents-do-with-the-zkagi-solana-agent-kit}

<p className="snippet-answer">The ZkAGI x Solana Agent Kit extends the Solana Agent Kit to let any agent, using any model, autonomously perform 15+ Solana actions. These include trading tokens, launching tokens, lending assets, sending compressed airdrops, and executing blinks.</p>

ZkAGI x Solana Agent Kit extends the Solana Agent Kit to let any agent, using any model, autonomously perform **15+ Solana actions**:

- Trade tokens
- Launch new tokens
- Lend assets
- Send compressed airdrops
- Execute blinks

## How does the PrivacyCash SDK enable private transactions for agents? {#how-does-the-privacycash-sdk-enable-private-transactions-for-agents}

<p className="snippet-answer">The PrivacyCash SDK enables private deposits, withdrawals, and balance checks directly from Solana Agent Kit agents. This allows agents to handle funds securely and anonymously, unlocking autonomous, privacy-preserving, and verifiable AI agents on Solana.</p>

### Why It Matters for Agents

- **Confidential Treasury Management** — Agents can hold funds without leaking wallet activity to competitors
- **Unlinkable Payouts** — Agents can pay out or move assets without exposing the funding source
- **Stealth Strategies** — Trading agents can operate without revealing transaction footprints

### Key SDK APIs

- `deposit()` — Move SOL into private UTXOs
- `withdraw()` — Send SOL privately to any address
- `getPrivateBalance()` — Check shielded balance

This unlocks **autonomous, privacy-preserving, and verifiable AI agents** on Solana.

## What is the future scope of ZkAGI's Solana Agent Kit? {#what-is-the-future-scope-of-zkagis-solana-agent-kit}

<p className="snippet-answer">Future plans for the ZkAGI Solana Agent Kit include cross-chain privacy bridges, programmable privacy rules for agents, and direct agent-to-agent confidential payments. Integration with Ethereum, Aptos, Zcash, and DAO treasury privacy are also on the roadmap.</p>

- **Cross-chain Privacy Bridges** — Integrate private transactions across Ethereum, Aptos, and more
- **Programmable Privacy** — Agents define rules on when and how funds are revealed
- **DAO Treasury Privacy** — Private management for collective governance funds
- **Agent-to-Agent Payments** — Direct peer-to-peer confidential payments between AI agents
- **Connect Solana Agents to Zcash**
