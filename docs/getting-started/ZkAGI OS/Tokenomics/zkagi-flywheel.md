The ZkAGI flywheel is comprised of the following actors:

- Retail/Instituitional Investors and the ZkAGI network together give $ZKAGI value

- GPU / CPU miners or Validators who are providing compute to network (ZkAGI Cluster) in order to generate Zk Proofs and run AI models are rewarded using $ZKAGI -Third Party Developers who develop on the Zynapse API to deploy AI agents, these AI agents use $ZKAGI to operate on the network (they consume compute and Zynapse API credits)

- Users who interact with agents/buy agent coins developed through the end user product ZkTerminal (formely called ZkSurfer) using $ZKAGI as the base currency. These AI agents on ZkTerminal give users memetic and conventional utility / returns

- Finally, the community of users and core + founding team make up the ZkAGI network, and the cycle of the ZkAGI economy continues.
