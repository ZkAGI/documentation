## How does the ZkAGI Flywheel work? {#how-does-the-zkagi-flywheel-work}

<p className="snippet-answer">The ZkAGI Flywheel is an economic cycle where investors give $ZKAGI value, compute providers earn $ZKAGI for running AI models and generating Zk proofs, and developers and users consume $ZKAGI to deploy and interact with AI agents. This self-reinforcing loop drives continuous growth of the ZkAGI network.</p>

- **Retail/Institutional Investors** and the ZkAGI network together give **$ZKAGI** value
- **GPU/CPU miners or Validators** provide compute to the network (ZkAGI Cluster) to generate Zk Proofs and run AI models, and are rewarded using **$ZKAGI**
- **Third-Party Developers** build on the **Zynapse API** to deploy AI agents that consume compute and Zynapse API credits, paid in **$ZKAGI**
- **Users** interact with agents and buy agent coins through the end-user product **ZkTerminal** (formerly called ZkSurfer) using **$ZKAGI** as the base currency
- These AI agents on ZkTerminal give users memetic and conventional utility/returns
- The community of users and core + founding team make up the **ZkAGI network**, and the cycle continues

## Who are the key participants in the ZkAGI economy? {#who-are-the-key-participants-in-the-zkagi-economy}

<p className="snippet-answer">The ZkAGI economy consists of four key participant groups: investors who provide token value, compute providers who supply GPU/CPU resources, developers who build on the Zynapse API, and end users who interact with agents via ZkTerminal. Together these groups form a self-sustaining economic flywheel.</p>

| Participant | Role | Economic Activity |
|---|---|---|
| **Retail/Institutional Investors** | Provide market value to $ZKAGI | Token investment and liquidity |
| **GPU/CPU Miners & Validators** | Supply compute to ZkAGI Cluster | Earn $ZKAGI for generating Zk Proofs and running AI models |
| **Third-Party Developers** | Build on Zynapse API | Deploy AI agents that consume compute and API credits |
| **End Users** | Interact via ZkTerminal | Purchase agent coins and use agents with $ZKAGI |
| **Community & Core Team** | Govern and grow the ZkAGI network | Sustain the economic cycle |
