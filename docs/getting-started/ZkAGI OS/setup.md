# Setting up ZkAGI OS on Your Device

There are 4 ways to access and run the ZkAGI OS to create agents and applications:

- **Browser Chat Terminal**: On Browsers (mobile & Desktop) with Phantom Wallet / metamask snap - Access ZkTerminal  to access the ZkAGI OS (chrome is recommended, available now)
- **Social Apps**: Using Tg app & associated X authorization (to be released soon)
- **Local/Remote Desktop or VPS**(partial release for compute sharing done, agent setup coming soon)
- **Programmatic Access through Zynapse API** (expected release Q2 2025)
