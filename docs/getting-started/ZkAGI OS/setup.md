## How do I set up ZkAGI OS on my device? {#how-do-i-set-up-zkagi-os-on-my-device}

<p className="snippet-answer">ZkAGI OS can be accessed through four different methods depending on your platform and use case. The most immediately available option is the Browser Chat Terminal using Chrome with a Phantom Wallet or MetaMask snap.</p>

There are 4 ways to access and run the ZkAGI OS to create agents and applications. Choose the method that best fits your workflow:

## What are the different ways to access ZkAGI OS? {#what-are-the-different-ways-to-access-zkagi-os}

<p className="snippet-answer">ZkAGI OS offers browser-based access via ZkTerminal, social app integration through Telegram and X, local/remote desktop or VPS deployment, and programmatic access through the Zynapse API. Browser access via Chrome is available now, with other methods rolling out progressively.</p>

- **Browser Chat Terminal** — Available on mobile and desktop browsers with Phantom Wallet or MetaMask snap. Access ZkTerminal to use ZkAGI OS (Chrome recommended, available now).
- **Social Apps** — Using the Telegram app and associated X authorization (to be released soon).
- **Local/Remote Desktop or VPS** — Partial release for compute sharing done; agent setup coming soon.
- **Programmatic Access through Zynapse API** — Expected release Q2 2025.
